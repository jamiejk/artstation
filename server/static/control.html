Pen height (Z)
Standard AxiDraw pen lowering and raising.
Live Z: —/100 (0 = map deepest · 100 = map highest). Jog with Z ↑/↓ or Shift+↑/↓ until motion stops, then Set Max Lift (clear of paper) and Set Max Lower (nib on paper).
Typical order: (1) Set Bed Top Left (2) Jog Z → Set Max Lift (clear of paper) and Set Max Lower (deepest drawing press) (3) Seat Pen — Z rises a few stops above max lower; mount the pen so the nib just leaves a mark (4) Optional: Run Pen Test Strip. Seating above max lower keeps press room below and lift room above for Z modulation.
